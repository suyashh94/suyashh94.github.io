World of Bits / MiniWoB (ICML 2017)

Benchmark for web‑based agents · ← Back to VLA

Summary

MiniWoB provides a suite of browser tasks (forms, buttons, etc.) with programmatic interfaces, enabling supervised and RL studies of low‑level UI interactions.

Key Contributions

  • Standardized, reproducible UI task set for RL.
  • Action space over DOM/low‑level events for fine‑grained control.
  • Baseline agents and evaluation protocols.

Notes

Use as a sandbox to debug action schemas and reward shaping before scaling to real sites.
