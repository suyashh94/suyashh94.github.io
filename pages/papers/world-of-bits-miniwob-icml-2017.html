World of Bits / MiniWoB (ICML 2017)

Benchmark for web‑based agents · ← Back to VLA

Briefing

World of Bits (WoB) is an open‑domain platform for web‑based agents, targeting realism by letting agents act with low‑level keyboard and mouse inputs on actual web tasks. It addresses reproducibility via cached HTTP traffic and uses crowdworkers both to author tasks (natural language questions) and to provide demonstrations.

Main Themes

  • Bridge the realism gap in RL by operating in web environments rather than purely synthetic games.
  • Enable research on web‑based RL with low‑level actions that mirror human interactions.
  • Ensure reproducibility on a transient web via HTTP caching and curated tasks.

Methodology

  • Crowdworkers create tasks defined by natural‑language questions.
  • Crowdworkers provide demonstrations using keyboard and mouse.
  • Cache HTTP traffic to approximate a stable, offline website snapshot.

Key Facts

  • Purpose: open‑domain platform to study realistic web tasks.
  • Action space: low‑level keyboard/mouse on web UIs.
  • Evidence: agents trained via behavioral cloning and RL succeed on multiple tasks.

Source: Shi et al., ICML 2017 (World of Bits).
